PWA ICON GENERATION INSTRUCTIONS
================================

You need to create three icon files for the PWA to work properly:

1. favicon.ico (32x32px or 64x64px)
2. icon-192.png (192x192px)
3. icon-512.png (512x512px)

Design Guidelines:
- Background: True black (#000000)
- Symbol: "MS" or "ST" initials (Mr. Sedo-Ta) in white/accent color, or a phone icon
- Text: "Mr. Sedo-Ta Portfolio" (for favicon context)
- Style: Minimal, modern, One UI inspired
- Format: PNG with transparency (or ICO for favicon)
- Square dimensions (no padding needed, system will add)

Quick Generation Options:
- Use Figma/Canva to design
- Use an online favicon generator
- Use an AI image generator with prompt: "minimal black portfolio icon with white P letter, modern, clean"

Tools:
- https://realfavicongenerator.net/
- https://www.figma.com/
- https://canva.com/

Once generated, replace the placeholder files:
- /public/icon-192.png
- /public/icon-512.png
- /public/favicon.ico
